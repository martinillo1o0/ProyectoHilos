#Hilos y sus derivaciones.

Este proyecto explora las distintas formas de trabajar con hilos en Java, incluyendo:

✅ Hilos heredados (Thread)

✅ Hilos con programación funcional (Runnable con lambdas)

✅ Herencia de Runnable (implementación clásica)

✅ Herencia de Runnable con programación funcional

Descripción
El objetivo es demostrar cómo se pueden utilizar hilos en Java para la ejecución de tareas concurrentes, utilizando tanto clases anónimas como expresiones lambda para una implementación más compacta y eficiente.



Que es un flag?
Es una variable que en los programas actua como un indicador y puede servir para pausar o detener el programa.

Que es el metodo wait()? Es un metodo que hace que un hilo espere hasta que le manden una notificacion
